# 项目扩展模块清单

## 说明
本文件用于记录当前项目额外启用的扩展模块。

规则：

1. 扩展注册不等于扩展激活
2. 只有在本文件或 `ACTIVE_STANDARDS.md` 中显式声明后，扩展才视为启用
3. 扩展只能增强当前节点，不得接管主 workflow
4. 自定义扩展在启用前必须先通过安全扫描
5. 扩展默认优先走“项目级卸载”，而不是直接删除包文件
