# 文件定位
面向营销落地页 / 产品展示页 / 活动专题页的场景规范包。

## 适用项目
- 营销落地页
- 产品宣传页
- 活动专题页

## 典型业务目标
- 传达产品价值、活动价值或品牌主张
- 驱动注册、预约、咨询、留资或下载转化

## 典型用户与角色
- 潜在客户
- 访客
- 市场 / 品牌运营
- 活动运营

## 核心模块构成
- Hero 主叙事区
- 价值卖点区
- 证明 / 案例区
- CTA 转化区
- FAQ / 补充说明区

## 页面/端形态特征
- 单页或少页结构常见
- 强视觉叙事、节奏感和转化引导明显
- 动效、媒体、排版和内容层级是重点

## 数据与流程特征
- 数据模型轻，内容结构、媒体资产和 CTA 路径更关键
- 表单留资、埋点、跳转和来源追踪常见

## 交付重点
- 视觉质感
- 叙事结构
- 转化路径清晰

## 高风险点
- 视觉吸引力不足
- 叙事层级混乱
- CTA 不明确或路径断裂

## 工作流附加约束
- Stage 2 必须先冻结叙事结构、版式节奏和 CTA 位置
- Step 5b / 5d 要重点校验动效、媒体与响应式表现
- 若有留资或预约动作，Step 5c 先冻结表单字段、回执与埋点

## 推荐配套技术栈方向
- Next.js / Astro / React
- 强视觉前端实现
- 静态发布或前端优先交付

## 验收重点
- 首屏吸引力和叙事节奏达标
- CTA 路径明确
- 响应式和动效表现稳定
