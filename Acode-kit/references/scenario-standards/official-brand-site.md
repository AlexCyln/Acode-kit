# 文件定位
面向官网 / 品牌站 / 营销官网的场景规范包。

## 适用项目
- 企业官网
- 产品官网
- 品牌活动页

## 典型业务目标
- 提升品牌表达、产品理解和线索转化

## 典型用户与角色
- 潜在客户
- 合作方
- 品牌访客

## 核心模块构成
- 首屏 Hero
- 产品 / 能力介绍
- 案例 / 证言
- 转化 CTA
- 联系 / 留资

## 页面/端形态特征
- 强视觉叙事
- 节奏化段落
- 明确 CTA 和信任构建模块

## 数据与流程特征
- 结构相对简单
- 转化链和埋点链重要

## 交付重点
- 首屏表达力
- 信息叙事顺序
- 转化路径清晰

## 高风险点
- 视觉漂亮但价值表达不清
- CTA 多而散

## 工作流附加约束
- Stage 2 先冻结视觉叙事结构和 CTA 主链
- Step 5b 要覆盖首屏、能力段、转化段关键状态

## 推荐配套技术栈方向
- React / 静态站点 / 营销页技术栈

## 验收重点
- 价值表达明确
- 转化路径顺畅
